## 🧪 How It Works

1. Users sign up and log in via the frontend.
2. Auth state is maintained using session storage.
3. Backend APIs handle:
   - Authentication
   - Project & domain data
   - Enrollments
   -submission of project
4. MySQL stores all user and project data.
